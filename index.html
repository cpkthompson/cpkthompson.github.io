This is my personal website and it's still relatively new so please be gentle. It’s also a playground for me so I’m constantly trying new stuff, making changes, and probably breaking things.

This site was created using HTML, CSS, and JavaScript using the Python Web Framework: Django; It was carefully hand-coded in PyCharm and hosted on Heroku because it was free. Deployed in a continuous integration fashion, because Facebook bragged about it and why not? This was done with the help of GitHub. The codebase is version-controlled by Git and also managed on GitHub. The type is set in lovely Karla and Raleway, all served by Google Fonts.
